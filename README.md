## 自動起動設定

以下のデーモンはsystemctlコマンドで自動起動設定（enable）をしておく。
自動起動設定はOS起動時に自動的に立ち上がるソフトウェアを指定すること。

- httpd
- mysqld
- chrony
- samba
- memcached


## 最後の設定

ここで一旦rebootして、すべて正常に起動できるかを確認する。

```bash
[root@centos8 ~]# reboot
```
